# Calculator Python Project

## 📌 Description
This is a simple calculator application developed using Python. 
The project performs basic arithmetic operations such as addition, subtraction, multiplication, and division.

This project was created as part of my CodSoft Python Internship tasks.

## 🚀 Features
- Addition of two numbers
- Subtraction of two numbers
- Multiplication of two numbers
- Division of two numbers
- User-friendly command-line interface
- Handles basic input and output operations

## 🛠️ Technologies Used
- Python 3
